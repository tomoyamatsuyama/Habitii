## なぜ習慣化アプリ
- 毎年毎年、今年はこれを続けようと思って続けられない
- https://www.ted.com/talks/matt_cutts_try_something_new_for_30_days

## このアプリでできること
- 日々の習慣を登録
- 今日やることを終えたら、スワイプしてスッキリ
- allのタブで全ての習慣を一覧化
- カレンダー機能で、いつやっていつやれなかったかを確認
- カレンダー機能で、やれなかった日も、やれたことに編集（別の日に二倍やったという言い訳が可能）
